# Проект Олимпиады по Веб-Разработке (OlympDev)

Этот проект представляет собой набор статических HTML/CSS шаблонов для олимпиады по веб-разработке. Шаблоны используют HTML5 и Tailwind CSS (через CDN). Также включено простое Flask-приложение для запуска.

## Структура Проекта

- **index.html**: Главная страница со статическим слайдером (каруселью) и описанием преимуществ.
- **login.html**: Простая страница входа (логин/пароль).
- **register.html**: Расширенная форма регистрации с различными типами полей.
- **profile.html**: Страница профиля пользователя с аватаром, статистикой и таблицей сравнения.
- **catalog.html**: Каталог товаров с поиском, фильтрами и модальными окнами (CSS-only) для быстрого просмотра.
- **item.html**: Детальная страница товара с галереей и отзывами.
- **404.html**: Кастомная страница ошибки "Страница не найдена".
- **style.css**: Файл стилей, содержащий переменные для светлой/темной темы и кастомные классы.
- **app.py**: Приложение на Python (Flask) для маршрутизации шаблонов.

## Как Запустить

### Через Flask (Рекомендуется)
1. Убедитесь, что у вас установлен Python и Flask (`pip install flask`).
2. Поместите все файлы в одну папку.
3. Запустите файл `app.py` командой:
   ```bash
   python app.py
   ```
4. Откройте браузер и перейдите по адресу `http://127.0.0.1:5000/`.

### Как Статический Сайт
Просто откройте файл `index.html` в любом современном браузере. Все ссылки настроены относительно друг друга.

## Редактирование Шаблонов

### Изменение Темы (Цвета)
Цвета определены в файле `style.css` через CSS-переменные.
- `:root` содержит цвета для **Светлой темы** (по умолчанию).
- `.dark` содержит цвета для **Темной темы**.
Чтобы изменить основной цвет фона, найдите `--bg-secondary` в блоке `:root`.

### Хедер (Шапка сайта)
Хедер находится внутри тега `<header>` в каждом HTML-файле. Если вы хотите изменить навигацию, вам нужно отредактировать этот блок в каждом файле: `index.html`, `profile.html` и т.д.
- Логотип и навигация выровнены по левому краю.
- Кнопки входа и переключения темы выровнены по правому краю.

### Карусель (Слайдер)
Находится в `index.html` в блоке с классом `carousel`. Слайды — это блоки `div` с классом `slide`. Чтобы изменить фоновое изображение, отредактируйте стиль `background-image` в HTML.

### Модальные Окна (Каталог)
В `catalog.html` модальные окна работают на чистом CSS с использованием псевдокласса `:target`. У каждого модального окна есть `id` (например, `modal-1`), и ссылка-карточка ведет на этот якорь (`href="#modal-1"`).

## Решение проблем

**Ошибка: TemplateNotFound: index.html**
Эта ошибка возникает, если Flask не может найти HTML файлы. В данном проекте `app.py` настроен так, чтобы искать шаблоны в текущей папке (`template_folder='.'`). Убедитесь, что `app.py` находится в той же папке, что и HTML файлы.
